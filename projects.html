Outwit, Outplay, Outlast

An Analysis of the Characteristics that Contribute to the Success of Contestants on Survivor

This project is currently in progress. It aims to determine what characteristics make the winners of Survivor successful. Is it their inherent characteristics, like age, gender, personality type, or is it solely based on their gameplay strategy and decisions made within the game. Or is it both?

Python was used for the bulk of this project, but since the dataset was given as R files, I used R to convert the .rda files to .csv files for use in Python. Within Python, I used many packages, including pandas, numpy, scikit-learn, matplotlib, and seaborn. To create the visualizations, I used D3 in Javascript.

This project is currently unfinished - I'm still working on it as I learn more, but see the current landing page for it here: link

COGS 108

An Analysis on the Risks that Lead to Heart Problems

This project aimed to answer the question: To what extent can we predict one's risk of heart disease based on blood pressure, cholestrol, cholestrol checkups, BMI, smoking, stroke, diabetes, physical activity, diet, alcohol usage, healthcare, access to doctor, general health, mental health, physical health, different walk, sex, age, education, income?

This was a group project for COGS 108: Data Science in Practice. Python was used for most of this project.

Find the project code on GitHub: github.com/karinaechen/cogs108_project
